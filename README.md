## Usage

### Pre Requisites


Then, proceed with installing dependencies:

```sh
yarn install
```

### Lint TypeScript

Lint the TypeScript code:

```sh
$ yarn prettier
$ yarn lint
```

### Test

Run the Mocha tests:

```sh
$ yarn test
```

### Run Application

```sh
$ yarn run_app
```

### Clean

Delete the artifacts,reports and cache:

```sh
$ yarn clean
```
